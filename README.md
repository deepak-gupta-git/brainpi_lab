# BrainPi Innovation Lab

A modern, high-tech 10-page web application for an Innovation Lab focusing on Robotics, Drones, and AI education. Built with a focus on speed, responsiveness, and futuristic design.

## Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Responsive & Custom Gradients)
* **Icons:** Lucide-React
* **Routing:** React Router Dom (v6+)
* **Design Pattern:** Layout-based architecture with reusable components.

## Key Features
* **Dedicated Pages:** Includes specialized sections for Robotics, Drones, AI & Coding, and Lab Infrastructure.
* **Glassmorphic UI:** A sleek, dark-themed interface with semi-transparent blurs and high-tech aesthetics.
* **Lead Generation:** Fully functional inquiry form ready for backend integration.
* **Interactive Gallery:** A masonry-style filterable gallery for showcasing workshop highlights.
* **Performance Optimized:** Utilizing Vite for lightning-fast HMR and build times.
